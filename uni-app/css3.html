# flex布局

  • 当一个元素使用了 display： flex 的时候就建立了一个布局容器

  • flex的布局是按照主轴和副轴来的， 默认的是从左到右的一个横轴row， 如果是coloum是从上到下的一个副轴。

  • justic-content 控制的是主轴 align-xxx 控制的是副轴 一般space-between用的比较多。当中的 元素用flex 1 获取所有的空白空间。

  • flex-grow 是空白区域 占多少，可以放大一个

    flex-shrink 是控制缩小多少， 为0 的时候就是不缩小

    flex-basis:xxx px 是在元素不指定宽度的时候，以这个做为缩放的起始大小。

    可以简化为 flex 1 0 100px; 只有的格式来写。

  • 容器对自己容器里怎么排列的标示有 justic-content， flex-direction。。。

  • 容器里的元素，可以对自己进行 flex-grow, flex-shrink , flex-basis 等操作。

  • 用css的选择器选到子元素进行设置，，同时子元素也可以做为容器，只要加上 加上 display：flex

  • 一个控件里面的文字也默认是一个元素，当控件成为flex容器的时候，文件也就按照容器规则在里面排序。

    例子


<!DOCTYPE html>
<html lang="en">

<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width,initial-scale=1.0">
<title>flex演示</title>
<style>
    body{
        padding: 0;
        margin: 0;
        height: 100vh;
        background-color: blue;
        display: flex;
        flex-direction: column;
        justify-content: space-between;
        align-items: stretch;
    }

    headbar{
        height: 60px;
        background-color: blueviolet;
    }
    main{
        flex: 1;
        background-color: burlywood;
    }

    foot{
        height: 60px;
        background-color: #ddd;
        border-top:solid 1px black;
        display: flex;
    }

    foot section{
        flex: 1;     //foot 里面的 section 都平均扩张 把空白处平分掉
        background-color: chartreuse;
        border-right: solid 2px #fff;
        display: flex; //section 自己也成为一个容器 里面的 h4 和 ul 按照 column-reverse来排列
        flex-direction: column-reverse;
    }
    
    foot section:last-child{
        border-right: none; // css选择器，选择到最后一个section不要边框
    }

    foot section h4{
        display: flex; //h4里面的文字 也是一个元素，当h4成为一个容器，就可以对文字进行居中
        justify-content: center;
        align-items: center;
    }

foot section ul{
    border: solid 1px #fff;
    border-radius: 10px;
    display: flex; // ul也成为了一个容器，这时候默认的row排序不行了，要用column。
    flex-direction: column;
    justify-content: center;
    align-items: center;

}

</style>
