# 闪光配件

不同的闪光配件也是为了眼睛和嘴唇上不同的catch light。

人物距离背景3.5米，背景会变灰。 灯在人左前上方位置。

# 八角 75

  • 软硬：75 不大不小的情况下，光不是很硬，也不是很柔和。
  • 分布：主要是胸以上，胸以下迅速暗下来。

face

hair

# 八角 150

  • 软硬：150的光要柔和很多，鼻子附近的阴影几乎没有了， 嘴唇cl要长很多，头发cl要宽柔和些。 75的对比度要高一些。
  • 分布：光线整个身体都有。 背景也亮了。

fullbody

face

hair

# 柔光箱 长方形 35x60

  • 软硬：比八角75 对比度还要高， 脸和胸的亮度更亮，手上亮度也高，嘴唇和眼睛的cl要更亮。
  • 分布：光线胸部和肩以上跟下面对比更明显

fullbody

face

# 柔光箱 长方形 30x120 竖着

  • 软硬：相对于横放， 竖的时候 竖的方向大面积，是柔光(所以下巴这里是从上到下比较肉)， 横的方向是硬光（鼻子旁边的阴影是硬光 比较硬）。 竖放的时候则相反。
  • 分布：cl要分布到两个手臂上，全身都比较亮因为是竖着放的. 嘴唇的cl会因为横放的时候比较长，竖放的时候比较短。

fullbody

face

# 柔光箱 长方形 30x120 横放

  • 软硬：相对于横放， 竖的时候 竖的方向大面积，是柔光(所以下巴这里是从上到下比较肉)， 横的方向是硬光（鼻子旁边的阴影是硬光 比较硬）。 竖放的时候则相反。
  • 分布：上身比较亮. 嘴唇的cl会因为横放的时候比较长，竖放的时候比较短。

fullbody

face

# 柔光箱 60x100 竖放

  • 特色： 主要是竖放，模拟自然光。
  • 软硬： 箱子比较大，这时候比较柔。
  • 分布： 整体都比较亮

fullbody

# 柔光箱 120x180 竖放

  • 特色： 箱子大，可以模拟窗户光。 用第2个灯做为fill light， 把暗的一边打亮。
  • 软硬： 光线很柔和因为是大箱子
  • 分布： 整个身体都很亮，在侧边的时候一边特别亮，暗的那边要是用fl， 横过来的时候，光会包围这身体。

# 小标准罩

  • 特色： 光线很硬，一般放在头顶，需要模特化好妆，会非常突出细节。
  • 软硬： 很硬的光线
  • 分布： 整体都比较亮

# 标准罩 蜂巢 猪嘴 门

  • 特色： 光线集中，猪嘴最集中，门可以改变大小
  • 软硬： 很硬的光线
  • 分布： 蜂巢三个等级，光线越来越集中，猪嘴最集中，门可以调四个门变化最大。

# 伞 85 银色 白色 透光

  • 特点： 银色的伞对比 八角75 光线以中心向外又层次， 八角的柔光箱光线比较平均

    • 白色的伞，光线分布的时候不像银色的全部集中在上半身，比较平均
    • 透明85的伞 跟 八角150的比， 八角150对眼睛的cl更好看，能量更多，而且透光伞会漏光到镜头上。

  • 软硬： 比较软，脸上的阴影比较柔。

  • 分布： 上半身，从中心向外

  • 软硬： 比较软，脸上的阴影比较柔。

  • 分布： 上半身，从中心向外

# 美人伞 65

  • 软硬： 比较软，脸上的阴影比较柔。

  • 分布： 上半身，从中心向外
