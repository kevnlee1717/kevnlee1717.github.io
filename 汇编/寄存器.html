# 寄存器

# 寻址方式

  • [x]直接寻址
  • [bx] 设置bx位0 然后 add bx,1/2 根据字节或者字递增寻址
  • [bx+x] 两个要操作的地址相隔x位，用这样的寻址，只用bx一个指针就可以操作两个地址
  • [bx+si/di] 设置两个明显的源和目标指针，用法跟上面差不多

# 通用寄存器

AX BX CX DX 用来进行计算，存储结果等

# 地址寄存器

都是用来寻址的，不同的寄存器加上偏移来对代码，数据，栈等来进行寻址，对内存进行读写操作。

  • 数据 ds + 【】

  • 栈 ss + sp

    sp 设置了栈的大小，比如设置为f (16 byte) 那么就可以push8次，因为每次push都是以字型为单位，push两个byte,虽然理论上是8次，但是实际上并不进行检查，多出来的push 和 pop 会发生越界。

    push 的时候 sp-2 然后push

    pop 的时候sp+2 然后pop

    • 栈的作用
      • 当函数跳转开始的时候保存ip， 返回的时候取回ip
      • 保存临时数据，把要保存的寄存器或者内存地址先push

  • 代码 cs + ip

    cs就是代码段开始的位置，但是不一定是程序开始执行的位置，伪指令start才是程序开始执行的地方，那么cs到start的这段内存空间，可以存放自己定义的数据，定义的数据里，可以拿一部分做为自己的栈。 栈ss设置为cs，sp设置为start-cs的字节数。

      ``` ini

  code segment


          dw 11,22,33,44,55,66,77,88  ;自定义的数据
          dw 0,0,0,0,0,0,0,0          ;可以做为栈

  start:  mov ax, cs  ;程序执行的地方
          mov ss, ax  ;设置栈段为 代码段开始地方
          mov sp, 16  ;设置栈大小为16个字


  code ends

  ```

WARNING

寄存器都是以字型数(16bit 2字节)来操作的,高地址存高位，低地址存低位，比如 1122h 存储后就是 22 11 在内存中。
