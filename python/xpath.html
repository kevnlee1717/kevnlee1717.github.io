# xpath

# 总结

  • xpath最初是在在 html 节点上

  • 先找到一个需要的节点 根据 div table 或者直接用 id 和 class 属性来定位

  • 查找的时候是根据找的节点里的属性 然后进行筛选

  • []相当于是函数的包围 {} 一样，开始自定义的筛选函数

  • @xx 说明是哪个属性， id class text 都是属性

  • contains写在[]里，相当于在自定义筛选函数里，调用了一个contains方法，带两个参数

# 57 []和@

  • []相当于是函数的包围 {} 一样，开始自定义的筛选函数

  • @xx 说明是哪个属性， id class text 都是属性

  • contains写在[]里，相当于在自定义筛选函数里，调用了一个contains方法，带两个参数

  • html.xpath(.//div[@id=“first”]/div[@class=“aaa”]/text() 查找id为first的div下面 所有class为aaa的div里面的文本

使用 contains进行选择


div = html.xpath(".//td[contains(text(), '男')]")
print(div)
print('----------------')

# [<Element td at 0x11005d0c8>, <Element td at 0x1100432c8>, <Element td at 0x110043808>]
# 输出三个节点

div = html.xpath(".//td[contains(text(), '男')]/text()")
print(div)
print('----------------')

# ['男', '男', '男']
# 加上 text（）后，输出这三个节点的text

div = html.xpath(".//td[contains(text(), '男')]/../td[contains(text(), 'haoren')]/text()")
print(div)
print('----------------')

# ['zhangwei@haoren.com', 'xiaoxian@haoren.com']
# 返回到上一级，然后再筛选出所有的td，并在这些td的text里找出包括

# 59 and or not

# ['男', 'zhangwei@haoren.com', 'yifei@haoren.com', '男', 'xiaoxian@haoren.com', 'meijia@haoren.com', '男']


div = html.xpath(".//tr[3 and not(contains(@class, 'one'))]")

# 61 [pos] position last定位


div = html.xpath(".//tr[3][contains(@class, 'one')]")
print(div)
print('----------------')
# [3]直接定位第3个并且加上一个class筛选

div = html.xpath(".//tr[3 and contains(@class, 'one')]")
print(div)
print('----------------')
# 直接定位的另外一种写法


div = html.xpath(".//tr[position() > 3]")
print(div)
print('----------------')

# position定位大于哪个位置的

div = html.xpath(".//tr/td[last()]/text()")
print(div)
print('----------------')
