# 命令

# docker

  • docker build -t name . 根据当前目录下的 Dockerfile 建立一个名字为 name的 image，注意有个 。

  • docker images 查看本机所有的images

  • docker rmi xxxxx 删除一个 ID为 xxxxx的 image ， 只要id的前五位。

  • docker run -p xxxx:xxxx image-name 运行一个image需要指定端口号

    • docker run -ti image-name bash 运行一个image用bash 控制台
    • docker run -rm image-name cmds 运行一个container从image， 执行好cmds后，container就停止掉
    • docker run -d image-name 后台运行这个container
    • docker run -link server-name 用link的方式跟一个主机通信
    • docker run --net network-name image-name 运行的时候加入一个网络，通常link的时候使用

  • docker ps 查看运行中的container

    • docker ps -a 查看所有的container 包括停止运行的
    • docker ps -l 查看最后一个停止运行的container

  • docker attach container-name 进入一个container

  • docker start/stop ps-id 在一个已经运行过的image，用ps可以看到它的id，可以用id来运行和停止，不用指定端口号

  • docker logs container-name 输出运行中的container里的记录信息

  • docker rm container-name 把一个运行过的container移除

  • docker commit stopped-container new-image-tag 对一个停止的container进行保存为一个新的image

  • docker port container-name 查看一个container的端口映射

  • docker cteate network xxx 建立一个private network

# docker-compose

  • docker-compose build 根据当前目录下的 docker-compose.yml文件，对主要的image进行建立，依赖的那些在 up的时候开始建立

  • docker-compose up -d xxx 运行定义的一个container

  • docker-compose stop 停止运行中的container
