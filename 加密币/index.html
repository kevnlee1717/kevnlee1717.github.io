# 钱包

  • 钱包就是一个地址，自己的钱包就是自己的地址，可以用来收发币。

  • 币不多的时候就放在交易平台的钱包里就可以。

  • 交易平台的钱包可以把币发到自己钱包的地址，就转入了自己的钱包。

  • 钱包可以是冷钱包，这样就存在u盘里，更安全，需要的时候转出一部分到热钱包。

# 币的购买和卖出

  • 用担保平台，直接可以用人民币购买，也可以直接卖出到各个国家的买家，把钱到国外的账号。但是需要跟买卖的人沟通，怎么付款等。

    • https://localbitcoins.com/ 但是有限额，额度越大 认证的也越厉害。

  • 用币币交易，直接从市场里买入卖出，速度快，不需要跟人沟通。

    • 法币交易就是用法定的币人民币，美金去购买币。 币币交易就是 两种加密币直接交易。 usdt是跟美元挂钩的恒定币，在所有的币都在跌的时候可以换成usdt来保值。
    • https://www.huobi.com/zh-cn/ 在火币网，可以先用法币交易，买入usdt，再用币币交易用usdt在交易市场购买需要的比特币，这时候不需要去跟人沟通，设定好买入多少后，很快就会自动从市场中购买，卖出的时候也是一样。
