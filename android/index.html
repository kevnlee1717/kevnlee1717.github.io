# activity是什么

  • activity是由一个java文件和一个可选的布局文件xml构成，java文件在app/src/main/java/ 目录下， 布局文件在app/src/main/res/layout/目录下
MainActivity.java 路径：app/src/main/java/com/example/study/MainActivity.java
MainActivity.java 是所有逻辑代码所在的地方

       super.onCreate(savedInstanceState);

  • activity的布局在res文件下，但是对activity的声明和控制是在manifest文件里。
路径 ： app/src/main/AndroidManifest.xml
所有的activity声明在这里， 启动的时候启动哪个activity等
这里有一个secondActivity的声明。
1

<manifest xmlns:android="http://schemas.android.com/apk/res/android"
    package="com.example.study">

    <application
        android:allowBackup="true"
        android:icon="@mipmap/ic_launcher"
        android:label="@string/app_name"
        android:roundIcon="@mipmap/ic_launcher_round"
        android:supportsRtl="true"
        android:theme="@style/AppTheme">
        <activity android:name="com.example.second.secondActivity"></activity>
        <activity android:name=".MainActivity">
            <intent-filter>
                <action android:name="android.intent.action.MAIN" />

                <category android:name="android.intent.category.LAUNCHER" />
            </intent-filter>
        </activity>
    </application>

</manifest>

# activity手动创建

  1. 创建一个类xxxActivity，继承自 AppCompatActivity，类的名字将用在 manifest 文件里对activity进行定义。

    继承自 AppCompatActivity,类的名字将用在 manifest 文件里对activity进行定义。

  1. 在res/layout下新建立一个 resouce layout file ,这个文件res_file_name的名字将自动成为一个 R.layou.res_file_name。 用来在Java类的oncreat方法里，实现布局文件和java类的关联。

res_file_name的名字将自动成为一个 R.layou.res_file_name

  1. 在类xxxActivity 中实现方法 oncreae 要实现参数只有一个为 @Nullable Bundle savedInstanceState 的这个函数。

    函数里调用 setviewcontent方法，里面的id就是 R.layout.res_file_name

要实现参数只有一个为 @Nullable Bundle savedInstanceState 的这个函数

函数里调用 setviewcontent方法，里面的id就是 R.layout.res_file_name

  1. 最后在manifest文件里定义一个activaty，名字就为类的全名。

  1. 所以类的名字主要用在manifest文件里引用（对activity在app中进行声明），布局文件名用在类中的oncreate函数引用(关联布局文件和类)

# 控件注册事件的两种方法

两种方法

  • 可在布局文件里添加一个button，button添加一个属性 onclick方法，在activity类中实现这个方法。


   public void openThirdActivity(View view) {

        Intent intent = new Intent(MainActivity.this, secondActivity.class);
        startActivity(intent);

    }

  • 在布局文件中给button设置一个id，然后在activity类中根据id找到这个button，然后给button注册一个onclick的方法。


    Button bt_open = findViewById(R.id.bt_open_activity);
    bt_open.setOnClickListener(new View.OnClickListener() {
        @Override
        public void onClick(View v) {
            Intent intent = new Intent(MainActivity.this, secondActivity.class);
            startActivity(intent);
        }
    });
