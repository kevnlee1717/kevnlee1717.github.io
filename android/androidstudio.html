# 注意大小写

  • 在输入类名，控件名等名字的时候，需要注意大小写，才会跳出正确待选项目

# 快捷键

  • 快速搜索，双击shift键可以进行搜索文件，快速定位文件

  • 快速修复，option+回车，快速显示修复错误，进行包导入，快速定义变量等。

  • 快速定位到布局文件, 按住 command键 点击concrete方法的id里面布局文件名就可以快速打开布局文件。

  • 复制一行 command + d

  • 快速添加大括号 command + shift + 回车

  • 快速注释 cmd + / , 整块注释 cmd + shift + /

  • 快速查找类 cmd + o

  • 快速打开各种类型的文件 cmd + shift + o

  • 快速建立activity，右键点开后输入 act 再按右方向 输入 em 可以建立一个空的activity

  • 快速定义一个全局变量 cmd + alt + f
